# rps-chat

## Purpose:
This is a simple chat app based off of a childhood game that allows users to select a room and chat with others as well as play games together.
## Made with:
* Javascript
* Node.js
* Express.js
* MYSQL
* Sequelize
* Bcrypt
* Socket.io
* Express Sessions
